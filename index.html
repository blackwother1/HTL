Funksjoner i app/nettleser

  • 5-stjerners vurdering
  • Chat mellom brukere
  • Profiler knyttet til org.nr (Brønnøysund)
  • Referansebilder og kompetanseoversikt
  • Reiseavstand (arealsirkel)
  • Timelister for fakturering
  • Kryssleie mellom håndverkere
  • Rapportsystem med filtre

For deg med for mye jobb

  1. Legg ut annonse med behov og tidsplan
  2. Velg fagfelt og antall håndverkere
  3. Motta timelister i appen for fakturering

For deg med for lite jobb

  1. Trykk på "Har for lite"
  2. Bla gjennom tilgjengelige annonser
  3. Start chat og avtal lønn og dato
  4. Send timelister via appen
